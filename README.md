# st3247-group-15-assignment-code
code for st3247 assignment: Simulation-Based Inference for an Adaptive-Network Epidemic Model

## Overview

This repository contains all code for reproducing the analysis in our report.
We estimate three parameters (β, γ, ρ) of an adaptive-network SIR epidemic model
using four inference methods: rejection ABC, regression adjustment, ABC-MCMC, and SMC-ABC.

All simulations use a fixed base seed (`base_seed = 2026`) for full reproducibility.

## Requirements

```
pip install -r requirements.txt
```

## Reproducing Results

Open and run `ST3247_Group15_Analysis.ipynb` from top to bottom.
All results, figures, and tables in the report are generated by this notebook.

Run all cells in order — each cell depends on the variables defined in previous cells.
